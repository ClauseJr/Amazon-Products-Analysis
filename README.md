# Amazon E-Commerce Products Analysis

## Project Objectives

This project is an end-to-end data analysis of Amazon India's e-commerce platform focusing on understanding the relationship between product pricing, discounts, and customer satisfaction. Using a dataset of over 1,400 products spanning categories such as Electronics, Computers and Accessories, and Home and Kitchen, the analysis explores how discounting strategies influence product ratings, which categories offer the best value for money, and what combination of price and quality makes a product a top deal. The project delivers actionable insights for both sellers and shoppers , helping sellers understand competitive pricing and discount benchmarks across categories, while helping shoppers identify the best value products based on a composite deal score that combines discount depth and customer rating. The cleaned and enriched dataset, along with the visual dashboard, forms the foundation for machine learning models that will predict product ratings and classify products into price tiers, enabling data driven decision making on the Amazon India marketplace.
